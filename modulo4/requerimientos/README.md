Criterio de aceptación 2

Criterio de aceptación 3
Las configuraciones iniciales contienen las fuentes a ser usadas
Criterio de aceptación 4

Margin: 0;
Padding: 0;

Background-color: #fff;
Color: #555;
Font-family: ‘Lato’, ‘Arial’, sans-serif;
